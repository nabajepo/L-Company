# L-Company – Web Application (Java REST API)

## Overview

L-Company is a **web application developed in Java** that allows users to create an account and log in to access a company directory.

Once logged in, users can view other members of the company and find their email addresses.  
This application can be useful when someone wants to contact a colleague but only knows their name.

The project demonstrates the use of:

- Java
- RESTful Web Services
- MySQL database
- Maven
- Apache Tomcat server

---

## Application Language

⚠️ The application interface and documentation are **in French**.

Please follow the instructions in the documentation file to run the project.

---

## Features

Users can:

- Create an account (email, name, password)
- Log in to the system
- View other members of the company
- Search for email addresses of colleagues
